# 🔁 C Language Programs – Loops & Number Logic

This repository contains basic C programs focused on **loops and number-based logic**.  
These programs help in building strong fundamentals in **iteration, conditions, and problem-solving**.

---

## 📌 Programs Included

### 1️⃣ Character Pattern (do-while)
Prints characters starting from 'a' and increases by 4 each step until 'z'.

---

### 2️⃣ Count Number of Digits
Counts the total number of digits present in a given number using loop logic.

---

### 3️⃣ Sum of First and Last Digit
Finds the first and last digit of a number and calculates their sum.

---

## 🛠️ Technologies Used

- C Programming Language  
- GCC Compiler  
- Standard Input/Output  

---

## ▶️ How to Run

1. Compile the program using GCC  
2. Run the output file in terminal  

---

## 🎯 Learning Outcomes

- Understanding loops (`while`, `do-while`)  
- Working with digits of numbers  
- Logical thinking and problem-solving  
- Basic arithmetic operations
